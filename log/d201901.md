## [01月11日(金)](#11) <a id="11"></a>

ふと思い立って、HTMLで書いてた曰記のMarkdown化をした。

```mk
HTML=$(shell ls d*.html)
MD=$(HTML:.html=.md)

all: $(MD)

%.md2: %.html
  pandoc -f html -t markdown_strict $< -o $@

%.md: %.md2
  ruby convert.rb $< > $@

.PHONY: clean

clean:
  rm -f *.md *.md
```

pandocで原型を作ってみたらわりときれいにいったので、それをRubyで修正。
