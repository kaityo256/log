Formuraを入れてみる。ここを参考に。 まずはhaskellのインストール。

brew install haskell-stack
git clone git@github.com:nushio3/formura.git
git clone git@github.com:nushio3/formura.git 
cd formura
stack setup --install-ghc
stack build --trace

初回実行は結構時間がかかる。と思ったら上記は古かった。あたらしいリポジトリはここだ。

git clone git@github.com:formura/formura.git
cd formura
stack install

あとはsampleとかで遊べる。
