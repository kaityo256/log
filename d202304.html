sudo apt update
                        sudo apt upgrade

してみる。あらためてインストールしようとしたが入らない。仕方ないので公式から最新版をダウンロードして入れる。

sudo apt-get purge --autoremove pandoc
                        https://github.com/jgm/pandoc/releases/download/3.1.2/pandoc-3.1.2-1-amd64.deb
                        sudo dpkg -i pandoc-3.1.2-1-amd64.deb
$ pandoc --version
                        pandoc 3.1.2

あっさり最新版になった。

04月03日(月)

大学院ガイダンス。出席者少ない。あと、学生証などの交付が混んでた。来年は午後開始が良いかも。

名古屋大学の方からのメールを見て、ふと田中ダンさんを思い出した。初めて出会ったのは物性若手夏の学校ではなかったかと思う。「ダン」という名前が印象的だったので覚えている。Swarm oscillatorという面白い系を研究しており、日本物理学会領域11の第一回若手奨励賞の受賞者でもある。福井大学を経て、名古屋大学情報科学研究科多自由度システム情報論講座に赴任したはず。

複雑ネットワーク上の結合振動子系のダイナミクスによると、中尾裕也さんの後輩であり、ということはおそらく蔵本由紀研究室出身ということになる。ということは佐々さんの後輩ということでもあるのか。亡くなったのは2009年12月1日らしい。優秀な方が若くして亡くなったのは残念。
