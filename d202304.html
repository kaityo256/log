自分でもやってみた

Macでのパスワード生成はpwgen。

brew install pwgen
                        pwgen 12

とかやると長さ12のパスワードを大量に生成してくれる。記号つきは-yオプション。

pwgen -y 12

Ubuntuではmkpasswdやろうとするとパスワード聞かれてしまう。WSLでもpwgenで良いや。

sudo apt install -y pwgen
