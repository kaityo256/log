Rubyのパッケージ管理、まだ良くわかってない。

bundle config set --local path 'vendor/bundle'

を実行すると、.bundle/configというファイルが作成される。中身は

---
BUNDLE_PATH: "vendor/bundle"

これがあると、

bundle install

とだけした場合でも、ローカルにインストールしてくれる。source .venv/bin/activateが不要なのは良さげ。

bundle installを実行すると、そこにあるGemfileを参照する。今回はHexaPDFを使いたいので以下のように記述する。

# Gemfile
source "https://rubygems.org"

gem "hexapdf", "~> 1.3"

.gitignoreには、vendorは追加するが、.bundleは入れない、すなわち.bundleはリポジトリ管理する。
